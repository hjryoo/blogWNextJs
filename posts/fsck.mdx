## About Linux - fsck(e2fsck)
- 리눅스 파일 시스템에 문제가 생기면 kernel에서 해당 파티션을 강제로 ReadOnly로 전환
  - 리눅스내 파일 시스템 검사 및 복구 명령어 : fsck (e2fsck)
    : 해당 명령어 사용시 꼭 해당 파티션을 unmount한 상태로 진행해야함.
    : 관리자 명령어로 실행해야 함.
   옵션 : 
   -f : 이상 유무에 상관없이 강제적으로 파일 시스템 확인
   -j : 저널링 파일 시스템을 대상으로 작업 할 때 필수
   -v : 점검 내역 상세 보기
   -p : 사용자의 의견을 묻지 않고, 자동으로 파일 시스템을 체크
   -y : 작업 중 질문을 자동으로 yes 처리함
   -n : 작업 중 질문을 자동으로 no 처리함
