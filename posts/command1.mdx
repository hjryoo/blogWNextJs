## About Linux - Command for status
서버 사양 확인하는 명령어
	- OS Bit확인
		$getconf LONG_BIT
 
	- OS 계열확인
		$cat /etc/issue
 
	- 상세 OS확인
		$cat /proc/version
 
	- CPU 정보 확인
		$cat /proc/cpuinfo
 
	- RAM 용량 확인
		$cat /proc/meminfo
 
	- DISK 용량 확인
	 	$df -h
