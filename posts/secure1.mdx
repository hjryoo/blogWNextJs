## About Linux - secure
리눅스 보안 점검 내용

- 경로 확인
  - pam_tally 경로 확인
    - find / -name "pam_tally*"
  - pam_tally 또는 pam_tally2 확인

- 파일 및 디렉터리
  - /etc/pam.d/common-auth

- 조치 내용(파일 수정)
  - 기관 정책에 맞는 계정 잠금 임계값 및 시간 적용
    - ex) 계정 잠금 임계값(3회) / 시간(30초)
    - auth<Tab>required<Tab>pam_tally2.so onerr-=fail even_deny_root deny=3 unlock=30
